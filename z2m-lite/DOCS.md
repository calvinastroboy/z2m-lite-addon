# Z2M Lite Panel

A simplified, mobile-first Zigbee2MQTT management interface.

## Auto-Discovery

This add-on automatically discovers the Zigbee2MQTT add-on running on your system. No configuration needed!

If auto-discovery fails (e.g., you run Z2M on a separate machine), you can manually configure:

| Option | Default | Description |
|--------|---------|-------------|
| `z2m_host` | *(auto)* | Z2M frontend host |
| `z2m_port` | `8099` | Z2M frontend port |
| `z2m_path` | `/api` | Z2M WebSocket path |
