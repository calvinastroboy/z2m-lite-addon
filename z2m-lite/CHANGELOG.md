# Changelog

## 1.0.0

- Initial release
- Device overview with real-time status grid
- Device control: toggle, brightness, color temperature
- Multi-endpoint support (multi-gang switches)
- Group management: create, delete, add/remove devices
- Room management with local storage
- Device rename
- Pairing mode (permit join)
- Auto-discovery of Zigbee2MQTT addon
- WebSocket proxy (no direct Z2M access needed from browser)
- Ingress support for HA sidebar
- Traditional Chinese UI
